> 👾 Discord simple bot to understand how to create a bot using discord lib

## Download

```sh
git clone https://github.com/peguimasid/Discord-Bot.git
```

## Install dependencies

```sh
yarn
```

## Config

- Create a `config.json` file

```
{
  "BOT_TOKEN": "<YOUR_TOKEN_HERE>"
}
```

## Usage

```sh
node index.js
```
